# MTB-UNIS-config

Nástroj pro snadnou konfiguraci modulů MTB-UNIS na sbernici MTB.

Program vzikl na nastavování poloh serv. Postupně se přidávají další funkce na nastavení MTB modulu UNIS

## V čem je to vytvořeno

Qt creator, Qt6.
